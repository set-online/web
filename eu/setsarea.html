Europa Hegoaldea turistifikazioaren aurka

Turistifikazioaren aurrean dauden hirien sarea

SET sarea Europa Hegoaldeko kolektibo, elkarte eta gizarte-mugimenduen arteko sarea da, turismoaren ondorio sozial, ekonomiko eta ekologikoei aurre egiteko sortutakoa.

Hasi irakurtzen Sorrera manifestua (2018) Topaketak Wikipedian
Irudia: Wikimedia Commons

Zer da SET sarea?

Definizioa

2018tik aktibo dagoen kolektibo, elkarte eta gizarte-mugimenduen sarea da. Bere helburua da turistifikazioaren ondorio negatiboak salatzea eta horien aurka borrokatzea Europa Hegoaldeko hiri eta lurraldeetan, ekintza koordinatuen, proposamen politikoen eta sentsibilizazio-kanpainen bidez.

Ideiak eta aldarrikapen nagusiak

Ikusi Wikipedian

Honako hauek dira SET sarearen ideiak eta aldarrikapen nagusiak:

  • Turistifikazioaren kritika: Industria turistikoak auzo eta hiriak merkantzia bihurtzen dituela, tokiko biztanleria lekualdatzen duela, etxebizitzaren prezioak garestitzen dituela eta ehun sozial eta kulturala hondatzen duela salatzen dute.
  • Hirirako eta etxebizitzarako eskubidearen defentsa: Turismoarekin lotutako higiezinen espekulazioa gelditzea, alokairu turistikoa arautzea eta bizilagunentzako etxebizitza eskubidea bermatzea eskatzen dute.
  • Ekonomia dibertsifikatua: Turismoaren monokultiboarekiko dependentziaren aurka daude, eta tokiko beharrak kontuan hartzen dituzten, orekatuak eta inklusiboak diren eredu ekonomikoen alde egiten dute.
  • Justizia soziala eta ingurumenekoa: Turismoaren ingurumen-inpaktuak nabarmentzen dituzte (aztarna ekologikoaren hazkundea, ekosistemen narriadura, baliabideen kontsumoaren igoera), eta krisi klimatikoaren aurrean ingurumenaren babesa, elikadura-subiranotasuna eta erresilientzia lehenesten duten politikak aldarrikatzen dituzte.
  • Autoantolaketa eta nazioarteko elkartasuna: Hegoaldeko Europako hirien arteko elkarlan sareak sustatzen dituzte, esperientziak eta estrategiak partekatzeko, alternatibak oinarritik eraikitzeko eta tokiko komunitateen ahalduntzea indartzeko.

Garai nagusiak

Hiru aldi azpimarragarri

SET sareak bere jarduera areagotu du zenbait unetan, iritzi publikoan turistifikazioaren inguruko eztabaida indartuz eta hauspotuz. Hiru aldi azpimarragarri daude:

2017–2018

Harremanak ehundu eta sarea sortu

Turismo-jardueraren gorakadaren eta turistifikazioaren ondorio kaltegarrien aurrean, Europa Hegoaldeko hainbat herritan protesta-plataformak eta elkarteak sortu ziren. Harremanak ehuntzen hasi ziren, eta SET sarea sortu zen.

2020–2021

Pandemiaren kolpea eta eredua kolokan

COVID-19 pandemiak turismoaren gaineko ekonomi eredua kolokan jarri zuen, eta SET sareak egoera hori baliatu zuen sektorearen ahulezia salatzeko.

2024–2025

Mobilizazioen jauzia

2024an mobilizazioetan jauzi kualitatibo eta kuantitatibo bat eman zen, inoizko parte-hartze zabalena eta anitzena izanik. Kanariar Uharteetan hasi eta Mallorcan, Bartzelonan, Kantabrian, Donostian, Alacanten, Menorcan, Cadizen eta Granadan jarraipena izan zuten.

Manifestu fundazionala

Laburpen grafikoa

Hona hemen manifestuaren ideia nagusiak modu bisual eta sintetikoan. (Testu osoa estekan.)

1/6

Etxebizitza eskubidea

Bizilagunen kanporaketa, prezioen igoera eta auzoen “merkantilizazioa” salatu.

2/6

Hiria komunitate gisa

Auzo-bizitza, tokiko ekonomia eta zaintza-sareak lehenetsi, ez “kontsumo-eszenatoki” gisa.

3/6

Turismo-monokultiboaren kritika

Mendekotasun ekonomikoa, lan-prekarietatea eta “hazkunde infinituaren” logika problematizatu.

4/6

Trantsizio ekologikoa

Ingurumen-inpaktua eta baliabideen presioa murriztu: eredu jasangarriak eta erresilienteak.

5/6

Justizia soziala

Ondorioak ez dira neutroak: klase-arrakala, desberdintasunak eta eskubideen urraketa erdigunean.

6/6

Sare-lana eta ekintza

Hirien arteko koordinazioa, autoantolaketa eta presio soziala: ikusgarritasuna + alternatibak.
